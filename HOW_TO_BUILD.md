# Yogi Heisenbear - Build Guide

## Building the JAR

### On macOS/Linux:
```bash
chmod +x build-jar.sh
./build-jar.sh
```

### On Windows:
Double-click `build-jar.bat` or run:
```cmd
build-jar.bat
```

This creates `YogiHeisenbear.jar` - a runnable JAR file that works on any platform with Java installed.
